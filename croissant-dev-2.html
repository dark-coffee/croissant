Basic T-SQL Querying
This course provides a basic overview of SQL Server, relational database theory, database building blocks and an introduction into the T-SQL language and its syntax. The T-SQL section covers SELECT, FROM, WHERE, UPDATE, JOIN, and Transactions and their importance.
Additionally, this course also contains a practical exercise portion, to familiarise trainees with the course content.

Ad Hoc Workload Calculator
This script determines what percentage of the workload performed by the SQL Server is Ad Hoc.
The returned value is useful when determining whether to enable 'Optimize for Ad Hoc Workloads' when assessing performance.
The sys.configuration id for this value is: 1581.
